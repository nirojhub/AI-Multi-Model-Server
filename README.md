# AI-Multi-Model-Server

[![License: MIT](https://shields.io)](https://opensource.org)

FastAPI server, maps two different open-source local models to specific GPU devices, and binds them to unique endpoint URL

The API will be available at `http://localhost:8000`

Once running, visit below link to test the api

- Swagger UI: `http://localhost:8000/docs`
